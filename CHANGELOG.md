# Changelog

## [1.0.0] - 2025-11-25

### Added
- Initial release of Indian Currency Input Formatter
- Support for Indian number formatting (lakh/crore)
- Customizable currency symbols and formatting options
- Built-in validation for min/max values
- Support for both TextField and FormField widgets
- International number formatting support
- High-performance formatting with proper cursor handling

### Changed
- Updated dependencies to latest stable versions
- Improved documentation and examples
- Added comprehensive test coverage

## [0.1.0] - 2023-01-01

### Added
- Initial development version
- Basic currency formatting functionality
- Core text input handling
